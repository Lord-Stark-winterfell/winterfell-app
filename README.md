# Сайт футбольного клуба «Винтерфелл»

## Запуск

```bash
npm install
npm start
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:3000
```

Админ-панель открывается только по прямому адресу:

```text
http://localhost:3000/admin
```

Ссылки на админ-панель убраны из пользовательской части сайта, поэтому обычный посетитель её не видит.

## Доступ к админ-панели

По умолчанию:

```text
Логин: admin
Пароль: winterfell2026
```

Для публикации на домене пароль лучше задать через переменные окружения:

```bash
ADMIN_USER=admin ADMIN_PASSWORD=ваш_сложный_пароль npm start
```

Без авторизации нельзя изменить состав, матчи, новости и галерею: сервер проверяет токен администратора при каждой операции сохранения.

## Структура данных

Редактируемые данные хранятся в папке `data`:

- `players.json` — состав;
- `matches.json` — матчи;
- `news.json` — новости;
- `gallery.json` — галерея.

Изображения лежат в `assets/img`.
